Mean IO Stack

Summary

  • Pinterest style mean.io application that can be used as a knowledge base

Setup

  1. Followed steps in Docs
  2. Install npm and node.js -> Go here -> click install -> node-v0.10.28-x64.msi -> npm and node gets installed under C:\Program Files\nodejs and the path gets added to environment variables - Path
  3. Install bower: -> npm i -g bower -> (gets copied under C:\Users\IBM_ADMIN\AppData\Roaming\npm)
  4. Install MongoDB: -> mongodb -> installed under C:\Program Files\MongoDB 2.6 Standard
  5. Download and install git from here. -> installed under C:\Program Files (x86)\Git -> add C:\Program Files (x86)\Git\bin; to Path environment variable
  6. npm install -g meanio@latest
  7. mean init myPinterest
  8. cd myPinterest
  9. npm install
  10. create c:\data\db folder, this is where the default mongodb database will be created
  11. start mongodb with command -> mongod
  12. install grunt globally -> npm install -g grunt-cli (http://mean.io/#!/support#faq)
