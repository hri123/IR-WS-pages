WS1

Summary

  • learning dojo
  • learning spring over OSGi
  • learning WTi concepts
  • learning end to end web development from dojo to osgi to spring to hibernate

Setup

  1. import the project into eclipse
  2. set the target platform
  3. derby database table is required to run the viral patel example (script in the - 'repository' folder)
  4. change the start level of this viral patel plugin to 5 and the start level of osgi.services to 1
  5. dojo toolkit must be available to run offline
